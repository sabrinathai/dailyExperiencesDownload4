iPhone/iPad version 1.0.0

iOS10 and iOS11 Version

Install iOS version 1.0.0 for iOS10 and iOS11

iOS9 Version

Install iOS version 1.0.0 for iOS9

After downloading the app, go to Settings > General.
Scroll down to Device Management, and then click on it.
You will see Embodied Social Cognition.
Click on Embodied Social Cognition and then click on Trust "Embodied Social Cognition".

Android version 1.0.0

Install Android version 1.0.0

After downloading the app, go to Settings > Applications.
Select Unknown sources and then Verify apps.

Check that a data file has been created for you here!
